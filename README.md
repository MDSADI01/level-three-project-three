Vehicle Rental System - Database Queries

Project Overview 

This project demonstrates a simple vehicle rental system database design using PostgreSQL. The system include three main table :
1. Users : Stores customer and admin information.
2. Vehicles : Stores informmation about vehicles available for rent.
3. Bookings : Tracks vehicle bookings by users.

The Project also show table creation, constraints,relationships(1-to-many,many-to-one) and sample SQL queries to retrieve and analyze data.
